# PetShop Manager

Sistema de cadastro de clientes para pet shop.

## Status do Projeto
**Versão 1.0** - Cadastro de clientes funcionando.

## Como executar
1. Clone o repositório
2. `python -m venv myvenv`
3. `myvenv\Scripts\activate`
4. `pip install streamlit`
5. `streamlit run app.py`
